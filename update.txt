create a new repository on the command line:-
echo "# The-Covid-Ease" >> README.md
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/banerjeePrayas/The-Covid-Ease.git
git push -u origin main
